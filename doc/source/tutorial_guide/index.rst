Tutorial Guide
**************

We think, that the easiest way to learn the key concepts of the balder testsystem is learning-by-doing. For this, the
following example will introduce you to the most important components of the balder system, by testing an example
project. This example was created especially for balder. It represents a situation you can find in many applications.
